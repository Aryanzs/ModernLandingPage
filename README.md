## ⚡ Overview

Welcome to **Studio**, your gateway to next-generation AI-powered web design. This repository contains the source code for a premium, high-performance web application designed to showcase intelligent, automated website building capabilities.

### ✨ Features
- **Ultra-Fast Performance**: Built with Vite and React for lightning-fast loads.
- **Premium Aesthetics**: Liquid glass design, immersive video backgrounds, and buttery smooth Framer Motion animations.
- **Responsive & Modern**: Tailored with TailwindCSS for beautiful experiences on every device.
- **Optimized for Scale**: Configured with intelligent chunk splitting for seamless deployment and caching.

## 🚀 Getting Started

To get a local instance of Studio running, follow these simple steps.

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- Optional: Set up an `.env.local` to securely test backend integrations.

### Installation

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone <your-repository-url>
   cd new-webdesign-ai
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
